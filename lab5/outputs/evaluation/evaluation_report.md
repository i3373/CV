# Оценка качества сгенерированных изображений

Оценивались 36 изображений из `script/outputs/generated` и 25 исходных фотографий из `me`. FID и Inception Score не использовались.

## Выбранные метрики

1. **ReferenceSimilarity**  
   Сравнивает CLIP image embedding сгенерированного изображения с CLIP image embeddings исходных фотографий. Для каждой картинки берется среднее top-3 похожестей к референсам, затем считается среднее по всем генерациям. Метрика выбрана вместо CLIPScore, потому что LoRA обучалась на конкретного человека, и здесь важнее проверить сохранение внешнего сходства с исходными фото, а не только соответствие текстовому промпту.

2. **Sharpness, variance of Laplacian**  
   Считает дисперсию Лапласиана на grayscale-изображении. Чем выше значение, тем меньше размытие и тем больше локальных деталей. Метрика выбрана, потому что для портретной генерации важны четкость лица, глаз, контуров и мелких деталей.

3. **Diversity in CLIP image space**  
   Для изображений одного промпта считается среднее попарное расстояние `1 - cosine similarity` между CLIP image embeddings. Метрика выбрана, чтобы проверить, не дает ли модель почти одинаковые варианты при разных seed.

## Итоговые значения

| Метрика | Среднее значение | Интерпретация |
|---|---:|---|
| ReferenceSimilarity | 0.6701 | Сгенерированные изображения в среднем заметно похожи на исходные фото. |
| Sharpness | 479.99 | В среднем изображения достаточно детализированные, без сильного общего размытия. |
| Diversity | 0.1336 | Варианты внутри одного промпта отличаются умеренно, но остаются близкими по смыслу. |

## Вывод

ReferenceSimilarity показывает, что промпты с токеном `i3373_person` в среднем лучше сохраняют сходство с исходными фотографиями, чем class-only промпты. Самые высокие значения получились у `token_city`, `token_beach` и `token_forest`. Это ожидаемо: именно token-промпты активируют обученную LoRA-персонализацию.

Резкость выше всего у `token_metal`, `token_cyberpunk` и `token_elf_city`, потому что эти сцены содержат много контрастных деталей. Diversity умеренная: модель генерирует не одинаковые изображения, но сохраняет общий портретный стиль и идентичность.

В целом качество можно считать приемлемым: LoRA сохраняет сходство с исходными фото, изображения достаточно четкие, а разные seed дают отличающиеся варианты без сильного ухода от промпта.
